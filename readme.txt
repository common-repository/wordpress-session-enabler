== Description ==

This very small plugin help you to enable for use PHP Sessions in WordPress. (In default, WP sessions are disabled).



== Installation ==

* 1) Copy folder with plugin in your wp-content/plugins directory
* 2) Activate plugin
* 3) Now you can use $_SESSION superglobal array
